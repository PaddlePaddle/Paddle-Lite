# 支持模型

目前，Paddle-Lite 已严格验证 52 个模型的精度和性能。对视觉类模型做到了充分的支持，覆盖分类、检测和定位，也包含了特色的 OCR 模型的支持。对 NLP 模型也做到了广泛支持，包含翻译、语义表达等等。

除了已严格验证的模型，Paddle-Lite 对其他 CV 和 NLP 模型也可以做到大概率支持。
