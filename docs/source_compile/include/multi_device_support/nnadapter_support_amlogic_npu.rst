NNAdapter 支持晶晨 NPU
^^^^^^^^^^^^^^^^^^^^^^^^

.. list-table::

   * - 参数
     - 说明
     - 可选范围
     - 默认值

     - 是否编译晶晨 NPU 的 NNAdapter HAL 库

     - 设置晶晨 NPU SDK 目录
