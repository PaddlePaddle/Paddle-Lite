NNAdapter 支持华为昇腾 NPU
^^^^^^^^^^^^^^^^^^^^^^^^^^

.. list-table::

   * - 参数
     - 说明
     - 可选范围
     - 默认值
   * - nnadapter_with_huawei_ascend_npu
     - 是否编译华为昇腾 NPU 的 NNAdapter HAL 库
     - OFF / ON
     - OFF
   * - nnadapter_huawei_ascend_npu_sdk_root
     - 设置华为昇腾 CANN 目录

     - /usr/local/Ascend/ascend-toolkit/latest
