NNAdapter 支持华为麒麟 NPU
^^^^^^^^^^^^^^^^^^^^^^^^^

.. list-table::

   * - 参数
     - 说明
     - 可选范围
     - 默认值
   * - nnadapter_with_huawei_kirin_npu
     - 是否编译华为麒麟 NPU 的 NNAdapter HAL 库

     - 设置华为 HiAI DDK 目录
