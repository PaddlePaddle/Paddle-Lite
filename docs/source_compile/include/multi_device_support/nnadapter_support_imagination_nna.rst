NNAdapter 支持颖脉 NNA
^^^^^^^^^^^^^^^^^^^^^^^^

.. list-table::

   * - 参数
     - 说明
     - 可选范围
     - 默认值
   * - nnadapter_with_imagination_nna
     - 是否编译颖脉 NNA 的 NNAdapter HAL 库
     - OFF / ON
     - OFF
   * - nnadapter_imagination_nna_sdk_root
     - 设置颖脉 imgdnn SDK 目录
     - 用户自定义
     - 空值

详细请参考 `颖脉 NNA <https://www.paddlepaddle.org.cn/lite/develop/demo_guides/imagination_nna.html>`_
